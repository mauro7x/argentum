Algunas convenciones para el desarrollo:

En cuanto al case utilizado:
    - Nombres de las clases: PascalCase, ej. class UnaClase {};
    - Nombres de los métodos: definir lo que prefieran entre camelCase.
    - Nombres de las variables: snake_case, ej. UnaClase una_clase;

Configuración del estilizador de VS Code:
En la extensión de C/C++, ir a configuración y buscar los siguientes campos:
    - C_Cpp: Clang_format_fallback Style. Completar con el siguiente texto para que
    a todos nos estilice igual:
    { BasedOnStyle: Google, IndentWidth: 4, TabWidth: 4, AllowShortIfStatementsOnASingleLine: false , AllowShortFunctionsOnASingleLine: Empty }
    Avisar cuanto antes en caso de cambiar algo!

En search settings buscar:
    - Editor: Format On Save. Marcar la casilla para que se aplique el estilizador 
    cuando guarden el archivo. Ahorra tiempo!

Sientanse libres de modificar y/o agregar cualquier convención acá, avisen cuanto antes así escribimos código parecido :)
