[*] --> Accepter
Accepter --> ClientLogin
ClientLogin --> NewConnection
NewConnection --> ClientConeection
ClientConeection --> FinishedConnection
FinishedConnection --> [*]

Accepter: Acceptador del cliente
ClientLogin : Validación del cliente
NewConnection : Nueva conexión del cliente
ClientConeection : Comunicación con el servidor
FinishedConnection : Cliente disconectado
